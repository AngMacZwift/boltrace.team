Live Streaming with OBS

With OBS Studio, we have a branded template for use in live streams. This will include sponsorship details, and use BRT fonts, themes, and colours. The template will include BRT Orange and Hellcatz Purple specific variations, as well as a variation to include a web camera feed.

Required Tools

  • OBS Studio
  • OBS Shader Filter Plugin
  • OBS Template for BRT

Recommended Streaming Platforms

  • Restream.io
  • Mixer.com
  • Twitch.tv
  • YouTube

Restream.io will forward your stream to multiple platforms, and generally provides better quality forwarding your stream to YouTube than streaming directly to YouTube.

Using The Template

After installing OBS Studio, install the plugin above by copying the files into the OBS installation directory, directions available in their README. The plugin is what will provide the borders with rounded corners. Also install the provided fonts in the template package.

  1. Unzip the OBS.zip file you downloaded to somewhere useful, such as your Documents folder.
  2. Install the fonts under "OBS\Sources".
  3. Import the OBS scene collection: Scene Collection > Import, choose "OBS\BRT Scenes - OBS.json".
  4. Under Scenes, pick either "Zwifting Hellcatz Purple" or "Zwifting BRT Orange" as scene to use.
  5. Under Sources, update BRT Text, Powered By, BRT Logo, and Background Bolts to point to image files under "OBS\Sources".
  6. Add your audio sources (optional, recommended to at least have Zwift sounds).
  7. If using a webcam, update the source, use Alt+drag to crop the source to trim the view, so it's not too big; otherwise delete the webcam source.
  8. Choose method of game capture from available sources: Game Capture/Display Capture/Window Capture; the others can be removed.
  9. Double-click the "Name Text" source and update with your name.
